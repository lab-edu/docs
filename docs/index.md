# lab-edu 文档中心

面向核心开发成员的项目知识库，用于沉淀技术决策、阶段规划与实施规范，汇集系统架构、API 与部署 / 使用指南，助力快速上手与团队协作。

## 文档结构

- [开发指南](guides/index.md): 日常开发、测试、部署与协作流程。
- [阶段规划](phases/index.md): Phase 0 到 Phase 4 的路线图与实施内容。
- [架构设计](architecture/index.md): 系统边界、模块职责与技术方案。
- [参考资料](reference/index.md): 高复用规则与速查信息。

## 建议阅读顺序

1. 先读 [阶段规划](phases/index.md) 明确当前阶段目标。
2. 再读 [架构设计](architecture/index.md) 对齐系统边界与约束。
3. 开发前查 [开发指南](guides/index.md) 与 [参考资料](reference/index.md)。
